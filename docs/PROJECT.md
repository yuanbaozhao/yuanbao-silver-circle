# 元宝的银发圈 Project OS

> 本文件是项目唯一入口。任何人或 AI 在开始网站、内容、数据库、百科、研究或商业化工作前，必须先阅读本文件，再按任务读取对应制度文件。

## 一、项目使命

把中国银发经济中分散的公开信息，持续整理为系统、可信、可检索、可关联、可辅助决策的知识基础设施。

## 二、长期定位

“元宝的银发圈”不是个人博客，也不是新闻搬运站，而是逐步建设为：

1. 中国银发经济知识平台
2. Silver Wiki 银发百科与知识图谱
3. 企业、人物、政策、产品、案例和投融资数据库
4. AI 行业研究与问答平台
5. 行业资源连接与合作平台
6. 元宝在银发经济与 AI 领域的长期代表作

## 三、十年目标

- 建立具有行业影响力的银发经济知识 IP
- 形成持续更新的行业知识图谱
- 建立可信、结构化、可追溯的行业数据库
- 形成研究报告、指数、地图、榜单、课程和咨询产品
